== Description ==

Xhanch - My Advanced Options (developed by [Xhanch Studio](http://xhanch.com/ "Xhanch Studio")) is a plugin that provides commonly used options for your WordPress but not provided by WordPress by default. Such as:

- Disable Wordpress auto P tag on post/page and comments
- Disbale WordPress texturize that will change/clean up/mess up your content's HTML code
- Enable shortcodes usage on text widgets
- Disable post/page revision history/logs to save your database storage
- and more to come ...

This plugin should be an essential plugin for your WordPress websites.
